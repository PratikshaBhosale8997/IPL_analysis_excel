# 🏏 IPL Analysis Dashboard (2008–2024)

> An interactive Excel dashboard uncovering match trends, team performance, and player stats across 17 IPL seasons.

---

## 📌 Overview
This project presents a comprehensive analysis of IPL matches from 2008 to 2024 using an interactive Excel dashboard. It enables year-wise filtering and visual exploration of team wins, toss decisions, venue statistics, and top performers.

---

## ❓ Problem Statement
Cricket analysts and fans often lack a single unified view to compare team and player performance across multiple IPL seasons. This dashboard aims to bridge that gap by consolidating key metrics into one interactive report.

---

## 📂 Dataset
- **Source:** Kaggle – IPL Complete Dataset (2008–2024)
- **Records:** Match-level data covering results, toss decisions, venues, and awards
- **Fields Used:** Match date, teams, toss winner, toss decision, match winner, Player of the Match, venue

---

## 🛠️ Tools & Technology
| Tool | Purpose |
|------|---------|
| Microsoft Excel | Dashboard creation, data cleaning |
| Pivot Tables | Data aggregation and summarization |
| Slicers | Year-wise interactive filtering |
| Excel Charts | Bar, donut, and horizontal bar visualizations |

---

## 💡 Key Insights
- 🏆 **Mumbai Indians** lead all-time wins with **144 victories**
- 🏏 **Fielding first** wins **54%** of the time after winning the toss
- 🥇 **AB de Villiers** tops the Man of the Match awards with **25 awards**
- 🏟️ **M Chinnaswamy Stadium** hosts the most matches among all venues
- 📅 **Rajasthan Royals** won the inaugural IPL 2008 season

---

## ✅ Results & Conclusion
The dashboard reveals that toss decisions significantly impact match outcomes, with teams choosing to field first holding a consistent edge. Mumbai Indians dominate overall performance, while veteran players like AB de Villiers and CH Gayle consistently earn top individual honors. This analysis can assist team strategists in making data-driven decisions.
